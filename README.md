# DSC-501-Spring-2022
## Utica University Masters of Data Science
### Introduction To Data Science Course
This 8-week course covers the data analysis process (CRISP-DM, KDD), probability and statistics, and answers the questions, What is Data? and What is Data Science? The course also covers exploration and analysis of data with Excel, Alteryx Designer, and R to apply to a final project. Bias, ethics, and social responsivity are covered as contemporary issues in data science.
